About This Project

This web application is a final project designed to showcase foundational skills in HTML, CSS, and JavaScript for creating interactive and content-rich web experiences.

It serves as a hub for exploring personal planetary astrology. Users can learn about the meaning of each personal planet in different zodiac signs, and also generate a personalized report by manually entering their own planetary placements.

Key Features:

  • Semantic HTML: The site is structured using modern HTML5 elements like <header>, <nav>, <main>, <section>, and <footer> for clear organization and accessibility.
  • Responsive CSS: Styling is applied using an external CSS file, leveraging Flexbox for flexible layouts and media queries to ensure the design adapts gracefully across various devices (desktops, tablets, and mobile phones).
  • JavaScript Functionality:
    • Event Handling: Responds to user interactions such as form submissions and navigation clicks.
    • DOM Manipulation: Dynamically updates page content, such as displaying personalized reports.
    • Form Validation: Includes client-side JavaScript validation for user input in the report generation form, providing immediate feedback.
    • Data Usage (localStorage): Utilizes localStorage to temporarily store user-entered planetary data, allowing it to be passed between the form page and the report display page.
  • Multi-Page Navigation: A clear and functional navigation system allows users to easily move between the Home, Generate Report, and About pages, as well as individual planet interpretation pages.
